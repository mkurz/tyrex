Configuration, Messages and Logging

{@link tyrex.util.Configuration} provides basic configuration for Tyrex based on the tyrex.config configuration file. Several Tyrex services rely on this configuration file.

The member variable {@link tyrex.util.Configuration#verbose} determines whether Tyrex should emit verbose information messages, which can be used for troubleshooting purposes. This class also provides access to a random number generator.

{@link tyrex..util.Logger}.

{@link tyrex.util.LoggerPrintWriter} implements a PrintWriter using a Log4J category. Connectors and other J2EE service providers require a PrintWriter in order to log messages, and rely on this object as an adapter for the Log4J category.

{@link tyrex.util.Messages} performs formatting of messages based on the current locale, using a message resource file. This class is often used by exceptions and logging messages to perform formatting based on an actual string contents that is obtained form a resource file, depending on the current locale.

Misc

{@link tyrex.util.FastThreadLocal} is an optimized implementation of {@link java.lang.ThreadLocal} which uses a background thread to remove stale thread entries, doing away with the inefficiencies of {@link java.util.WeakHashMap}.
